# Push Buttons

## Required parts

| Component | Quantity |
| :--------- | -------: |
| Push button | 2 |
| LED       | 1 |
| 10K ohm Resistor  | 2 |
| 330 ohm Resistor | 1 |
| Jumper wires | 7 |
